# Heyy! Welcome to my Card Flip Memory Game

 This is that classic memory game where you flip cards and hunt for matches!

Wanna play in like 2 minutes? Here’s the chill way:

 # step 1:
 Download – just hit that green Code ↓ button → Download ZIP
 # step 2:
 Get VS Code – grab it free from code.visualstudio.com
 # step 3:
 Open the game – unzip the file → drag the folder into VS Code
 # step 4:
 Boom! Play – open terminal (Ctrl + ) → type live-server` → watch the magic happen!

That’s it – start flipping & beat your own high score!
